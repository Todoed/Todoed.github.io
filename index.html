Turn any text into a task with just one right click.




Todoed is a tool, assisting the user as if everything useful; thought, typed and/or spotted on the web browser is a delegate-able todo item first.



One of our favourite current use cases of this app

Making chats more productive and accountable

Use this fastest and simplest to do app for assigning to do items to yourself and your team right from within the chats.

Some ways of using this product

  • Use Todoed to create todos/tasks instantly.
  • Use Todoed to delegate todos/tasks instantly, without needing to switch screens or apps.
  • Clip text from anywhere on the browser and convert it into a todo. Works on any text - gmail, fb, yammer, and absolutely any where on the browser.
  • Just drop down from the chrome extension to view your own todos in the simplest and fastest possible way.
  • Color code todos/tasks for quick categorisation.
  • Edit this text or task on Todoed anytime.
  • Full view page allows you to view todos delegated to and todos delegated by, and also search tasks right there.
  • Use Todoed as a the most unique bookmarking experience (Coming Soon)
  • Use Todeod to add to asana, readability, kippt, pocket, etc. (Coming Soon)
  • Do all of the above on the mobile and all other such devices (Coming Soon)

We are working very hard to turn this into an extraordinary tool for facilitating global productivity one day. This you see, is only the beginning.


By the way, have you ever felt an urge that you should create a to do list for yourself but haven't yet started? Download Todoed and you will find yourself started already.






When you have a thought or a task, you should be able to instantly not only share it as an instant message or an email, you should also be able to delegate it at the instance of sharing it.
